## Тестовые учётные записи

Для проверки всех ролей и функциональности используйте следующие учётные записи:

| Роль | Логин | Пароль | Что можно делать |
|------|-------|--------|------------------|
| **Администратор** | `admin` | `admin123` | Полный доступ: создание, редактирование, удаление книг, управление пользователями |
| **Модератор** | `moderator` | `moderator123` | Редактирование книг, управление рецензиями |
| **Пользователь** | `user` | `user123` | Просмотр книг, написание рецензий |
| **Гость** | (не требуется) | (не требуется) | Только просмотр книг |

> Все тестовые пользователи создаются автоматически при первом запуске приложения.

## Реализованный функционал

#### 1. Управление книгами
- Добавление книги (только администратор)
- Редактирование книги (администратор и модератор)
- Удаление книги (только администратор) с модальным окном подтверждения
- Загрузка обложки (с проверкой MD5 для предотвращения дублирования)
- Поддержка Markdown в описании книги

#### 2. Поиск и фильтрация (Вариант 3)
- Поиск по названию (частичное совпадение)
- Поиск по автору (частичное совпадение)
- Фильтр по жанрам (мультиселект)
- Фильтр по годам (мультиселект, динамически из БД)
- Фильтр по объёму страниц (от/до)
- Сохранение фильтров при пагинации

#### 3. Рецензии
- Просмотр всех рецензий на странице книги
- Добавление рецензии (аутентифицированные пользователи)
- Оценка от 0 до 5 с текстовым описанием
- Поддержка Markdown в тексте рецензии
- Защита от повторной рецензии от одного пользователя
-  Отображение средней оценки книги

#### 4. Пользователи и роли
- Регистрация новых пользователей
- Аутентификация (вход/выход)
- Запоминание пользователя (чекбокс "Запомнить меня")
- Три роли: Администратор, Модератор, Пользователь
- Разграничение прав доступа
- Управление пользователями через админ-панель (для администратора)

#### 5. Пагинация
- Не более 10 книг на страницу
- Сохранение параметров поиска при переключении страниц
